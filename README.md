# RJRB Product Website

Static product website for Importadora y Exportadora RJRB Ltda, showcasing:

- Tornillos por cajas with a searchable, foldable product catalogue.
- Casa portátil modular with an image gallery.
- WhatsApp and email contact actions for customer enquiries.

## Project Files

```text
index.html              Main website structure
styles.css              Visual design and responsive layout
script.js               Gallery, catalogue search, and foldable product lists
assets/docs/            Original PDF catalogue
assets/images/          Portable house images and screw product images
assets/images/screws/   Product images extracted from the screw PDF
```

## Editing Guide

- Change main page text in `index.html`.
- Change colors, spacing, gallery frame size, and mobile layout in `styles.css`.
- Change tornillo products, gallery images, search behavior, or foldable lists in `script.js`.
- Add new portable house photos into `assets/images/`, then add them to `galleryItems` in `script.js` and the thumbnail list in `index.html`.
